- v1.0(26.03.17)
    ✨ 이 툴의 특징 (Notepad보다 좋은 점)
     > 실시간 자동 저장 (localStorage): 갑작스러운 세션 종료나 창 닫힘이 발생할 수 있습니다. oninput 이벤트를 통해 한 글자 적을 때마다 브라우저 로컬 저장소에 즉시 저장하므로 안심하고 작성하실 수 있습니다.
     
     > 구조화된 리스트 관리: Notepad는 파일 하나하나를 찾아 열어야 하지만, 이 툴은 왼쪽 사이드바에서 여러 개의 '생각 덩어리'를 빠르게 전환하며 보고서의 목차를 잡기에 좋습니다.
     
     > 보고서 작성 최적화: 하단의 [텍스트 복사] 버튼을 통해 작성한 내용을 바로 한글(HWP)이나 PPT로 옮기기 쉽도록 만들었습니다.
